# Schema GRAPPA

Database MySQL relativo al sistema di provisioning e gestione servizi TLC.

## Introduzione

Lo schema `grappa` contiene tabelle e viste per la gestione completa del ciclo di vita dei servizi di telecomunicazione: anagrafiche clienti e fornitori, gestione apparati e infrastruttura di rete (fibra, VLAN, rack, datacenter), ordini e lavorazioni, fatturazione, ticket e segnalazioni, servizi cloud (CDL/VCD), migrazioni OLO, e molto altro.

Lo schema e' derivato dal file `mysql_schema.json` e documentato nei singoli file `grappa_{NOME}.json`.
